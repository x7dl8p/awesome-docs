Welcome to the getting started guide for our documentation site. Follow these steps to set up your environment and begin using our tools.

## Installation

To install our package, run the following command in your terminal:

```bash
npm install @our-company/package
```

## Configuration

After installation, you need to set up the configuration file. Create a file named `config.js` in your project root and add the following content:

```javascript
module.exports = {
  apiKey: 'YOUR_API_KEY',
  environment: 'production'
}
```

## Usage

Here's a basic example of how to use our package in your project:

```javascript
import { Client } from '@our-company/package';

const client = new Client();
const result = await client.doSomething();
console.log(result);
```

For more detailed information, please check out our Components and API Reference sections.

# Section 3

Welcome to the web development toolkit guide for our documentation site. Follow these steps to set up your environment and start building modern web applications with our development tools.

## Installation

To install our web dev package, run the following command in your terminal:

```bash
npm install @our-company/web-dev-kit
```

## Configuration

After installation, you need to set up the configuration file. Create a file named `config.js` in your project root and add the following content:

```javascript
module.exports = {
  port: 3000,
  buildDir: 'dist',
  environment: 'development'
}
```

## Usage

Here's a basic example of how to use our web dev kit in your project:

```javascript
import { WebBuilder } from '@our-company/web-dev-kit';

const builder = new WebBuilder();
await builder.buildProject('./src', './dist');
console.log('Build complete!');
```

For more detailed information, please check out our Components and API Reference sections.
